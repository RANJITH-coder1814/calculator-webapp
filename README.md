# calculator-webapp
A responsive calculator web application built using HTML, CSS, and JavaScript that performs basic arithmetic operations with a clean and modern UI.

A simple and responsive calculator built using HTML, CSS, and JavaScript.

## 🚀 Features

* Basic arithmetic operations
* Clear button
* Keyboard support
* Responsive layout

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run

Open `index.html` in your browser.
